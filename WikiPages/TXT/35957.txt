<br><br /><span style="color: #808080;"> Beginn Funktions-ID:  480 Modul-ID:83</span><br><br /><h3>Stempel: Erstellen</h3><p>Öffnet einen Dialog mit dem ein neuer Stempel angelegt werden kann
</p><br><br /><div style="text-indent: 20px;"><img src="/wiki/filemanager/images\Funktionskatalog\83_480.png" alt="" /><br /></div><br><br /><div style="text-indent: 20px;"><h4>Parameter der Funktion</h4></div><div style="text-indent: 20px;"><b>Reiter Allgemein: Organisation</b></div><div style="text-indent: 20px;"><p> ->  Die Organisation wird mit dem Stempel verknüpft. </p></div><div style="text-indent: 20px;"><b>Reiter Allgemein: Archiv</b></div><div style="text-indent: 20px;"><p> ->  Das Archiv wird mit dem Stempel verknüpft. Die verfügbaren Archive werden abhängig von der Organisation geladen</p></div><div style="text-indent: 20px;"><b>Reiter Allgemein: Benennung</b></div><div style="text-indent: 20px;"><p> ->  Bezeichnung des Stempels, diese wird in MyWork angezeigt.</p></div><div style="text-indent: 20px;"><b>Reiter Allgemein: Beschreibung</b></div><div style="text-indent: 20px;"><p> ->  Beschreibt den Stempel </p></div><div style="text-indent: 20px;"><b>Reiter Allgemein: Stempel sofort speichern</b></div><div style="text-indent: 20px;"><p> ->  Wenn der Stempel auf einem Dokument angebracht wird, wird dies sofort gespeichert. 
</p></div><div style="text-indent: 20px;"><b>Reiter Allgemein: Stempel aktiv lassen</b></div><div style="text-indent: 20px;"><p> ->  Der Stempel bleibt aktiv nachdem er auf einem Dokument angebracht wurde. Der Stempel kann also mehrmals hintereinander verwendet werden ohne diesen jedes Mal anklicken zu müssen
</p></div><div style="text-indent: 20px;"><b>Reiter Allgemein: Eindeutiger Stempel</b></div><div style="text-indent: 20px;"><p> ->  Dieser Stempel kann nur einmal pro Dokument genutzt werden.
</p></div><div style="text-indent: 20px;"><b>Reiter Allgemein: Nach dem anbringen zum nächstem Dokument wechseln</b></div><div style="text-indent: 20px;"><p> ->  Wenn dieser Stempel auf einem Dokument angebracht wurde wechselt MyWork automatisch zum nächsten Dokument
</p></div><div style="text-indent: 20px;"><b>Reiter Felder: Anlegen</b></div><div style="text-indent: 20px;"><p> ->  Legt ein neues Datenbankfeld an</p></div><div style="text-indent: 20px;"><b>Reiter Felder: Bearbeiten</b></div><div style="text-indent: 20px;"><p> ->  Ein Feld wird mittels eines Feldes bearbeitet</p></div><div style="text-indent: 20px;"><b>Reiter Felder: Löschen</b></div><div style="text-indent: 20px;"><p> ->  Entfernt das fokussierte Feld aus der Liste</p></div><div style="text-indent: 20px;"><b>Reiter Felder: Aktualisieren</b></div><div style="text-indent: 20px;"><p> ->  Aktualisiert die Ansicht der Felder. Änderungen werden evtl. erst nach Aktualisierung der Felder anzezeigt
</p></div><div style="text-indent: 20px;"><b>Reiter Indexdaten: Anlegen</b></div><div style="text-indent: 20px;"><p> ->  Legt ein neues Feld für Indexdaten an</p></div><div style="text-indent: 20px;"><b>Reiter Indexdaten: Bearbeiten</b></div><div style="text-indent: 20px;"><p> ->  Bearbeitet das fokussierte Indexdatenfeld mittels eines Dialogs</p></div><div style="text-indent: 20px;"><b>Reiter Indexdaten: Löschen</b></div><div style="text-indent: 20px;"><p> ->  Entfernt das fokussierte Indexdatenfeld aus der Liste.</p></div><div style="text-indent: 20px;"><b>Reiter Indexdaten: Aktualisieren</b></div><div style="text-indent: 20px;"><p> ->  Aktualisiert die Ansicht der Indexdaten. Änderungen an den Indexdaten werden evtl. erst nach der Aktualisierung der Liste angezeigt
</p></div><div style="text-indent: 20px;"><b>Reiter Darstellung: Hintergrundfarbe</b></div><div style="text-indent: 20px;"><p> ->  Auswahl der Farbe für den Hintergrund des Stempels</p></div><div style="text-indent: 20px;"><b>Reiter Darstellung: Rahmenfarbe</b></div><div style="text-indent: 20px;"><p> ->  Auswahl der Farbe für den Rahmen des Stempels</p></div><div style="text-indent: 20px;"><b>Reiter Darstellung: Textfarbe</b></div><div style="text-indent: 20px;"><p> ->  Auswahl der Farbe für den Text des Stempels</p></div><div style="text-indent: 20px;"><b>Reiter Darstellung: Rahmenbreite</b></div><div style="text-indent: 20px;"><p> ->  Einstellen einer Breite Für den Rahmen des Stempels</p></div><div style="text-indent: 20px;"><b>Reiter Darstellung: Schriftart</b></div><div style="text-indent: 20px;"><p> ->  Auswahl einer Schriftart</p></div><div style="text-indent: 20px;"><b>Reiter Darstellung: Schriftgröße</b></div><div style="text-indent: 20px;"><p> ->  Auswahl einer Schriftgröße</p></div><div style="text-indent: 20px;"><b>Reiter Darstellung: Layout</b></div><div style="text-indent: 20px;"><p> ->  Hier kann das Layout des Stempels erstellt werden</p></div><div style="text-indent: 20px;"><b>Reiter Darstellung: Vorschau</b></div><div style="text-indent: 20px;"><p> ->  Zeigt eine Vorschau des Stempels an. So wird der Stempel auch auf den Dokumenten aussehen.
</p></div><div style="text-indent: 20px;"><b>Reiter Gruppen: Hinzufügen</b></div><div style="text-indent: 20px;"><p> ->  Gibt den Stempel für weitere Gruppen frei
</p></div><div style="text-indent: 20px;"><b>Reiter Gruppen: Entfernen</b></div><div style="text-indent: 20px;"><p> ->  Entzieht der fokussierten Gruppe den Zugang zum Stempel
</p></div><div style="text-indent: 20px;"><b>Reiter Gruppen: Aktualisieren</b></div><div style="text-indent: 20px;"><p> ->  Aktualisiert die Ansicht der Gruppen
</p></div><p></p><span style="color: #808080;"> Ende Funktions-ID:  480 Modul-ID:83</span><hr/>
