<br><br /><span style="color: #808080;"> Beginn Funktions-ID:  665 Modul-ID:92</span><br><br /><h3>Bearbeiten</h3><p>Öffnet einen Dialog zum Bearbeiten einer Gruppe.
</p><br><br /><div style="text-indent: 20px;"><img src="/wiki/filemanager/images\Funktionskatalog\92_665.png" alt="" /><br /></div><br><br /><div style="text-indent: 20px;"><h4>Parameter der Funktion</h4></div><div style="text-indent: 20px;"><b>Gruppe</b></div><div style="text-indent: 20px;"><p> ->  Festlegen von Informationen zur Gruppe. Zum Beispiel Organisation, Gruppentyp und Verantwortlicher.</p></div><div style="text-indent: 20px;"><b>Aufgabenempfang aktivieren</b></div><div style="text-indent: 20px;"><p> ->  Aktiviert die Option, dass die Gruppe Aufgaben empfangen kann.</p></div><div style="text-indent: 20px;"><b>Benutzer: Hinzufügen</b></div><div style="text-indent: 20px;"><p> ->  Fügt der Gruppe weitere Benutzer mittels eines Dialogs hinzu.</p></div><div style="text-indent: 20px;"><b>Benutzer: Entfernen</b></div><div style="text-indent: 20px;"><p> ->  Entfernt den fokussierten Benutzer aus der Gruppe.</p></div><div style="text-indent: 20px;"><b>Benutzer: Aktualisieren</b></div><div style="text-indent: 20px;"><p> ->  Aktualisiert die Ansicht der Benutzer.</p></div><div style="text-indent: 20px;"><b>Profile: Hinzufügen</b></div><div style="text-indent: 20px;"><p> ->  Fügt der Gruppe weitere Profile mittels eines Dialogs hervor.</p></div><div style="text-indent: 20px;"><b>Profile: Entfernen</b></div><div style="text-indent: 20px;"><p> ->  Entfernt das fokussierte Profil aus der Gruppe.</p></div><div style="text-indent: 20px;"><b> Dokumentenstruktur: Hinzufügen</b></div><div style="text-indent: 20px;"><p> ->  Vergibt der Gruppe das Recht auf eine weitere Dokumentenstruktur mittels eines Dialogs.</p></div><div style="text-indent: 20px;"><b>Dokumentenstruktur: Entfernen</b></div><div style="text-indent: 20px;"><p> ->  Entfernt die fokussierte Dokumentenstruktur aus der Gruppe.</p></div><div style="text-indent: 20px;"><b>Archiv Dialoge: Hinzufügen</b></div><div style="text-indent: 20px;"><p> ->  Vergibt der Gruppe das Recht auf einen weiteren Archiv Dialog mittels eines Dialogs.</p></div><div style="text-indent: 20px;"><b>Archiv Dialoge: Entfernen</b></div><div style="text-indent: 20px;"><p> ->  Entfernt den fokussierten Archiv Dialog aus der Gruppe.</p></div><div style="text-indent: 20px;"><b>TFA Profile: Hinzufügen</b></div><div style="text-indent: 20px;"><p> ->  Vergibt der Gruppe das Recht auf ein weiteres TFA-Profil mittels eines Dialogs.</p></div><div style="text-indent: 20px;"><b>TFA Profile: Entfernen</b></div><div style="text-indent: 20px;"><p> ->  Entfernt das fokussierte TFA Profile aus der Gruppe.</p></div><div style="text-indent: 20px;"><b>Exportieren</b></div><div style="text-indent: 20px;"><p> ->  Exportiert die Gruppenliste.
</p></div><div style="text-indent: 20px;"><b>Löschen</b></div><div style="text-indent: 20px;"><p> ->  Löscht die fokussierte Gruppe.
