<br><br /><span style="color: #808080;"> Beginn Funktions-ID:  636 Modul-ID:88</span><br><br /><h3>Archiv: Hinzufügen</h3><p>Öffnet einen Dialog um ein neues Archiv anzulegen.
</p><br><br /><div style="text-indent: 20px;"><img src="/wiki/filemanager/images\Funktionskatalog\88_636.png" alt="" /><br /></div><br><br /><div style="text-indent: 20px;"><h4>Parameter der Funktion</h4></div><div style="text-indent: 20px;"><b>Archiv: Organisation</b></div><div style="text-indent: 20px;"><p> ->  Das Archiv wird dieser Organisation zugeordnet.</p></div><div style="text-indent: 20px;"><b>Archiv: Name</b></div><div style="text-indent: 20px;"><p> ->  Bezeichnung des Archivs.</p></div><div style="text-indent: 20px;"><b>Archiv: Beschreibung</b></div><div style="text-indent: 20px;"><p> ->  Anlegen einer Beschreibung für das Archiv.</p></div><div style="text-indent: 20px;"><b>Archiv: Speicherort</b></div><div style="text-indent: 20px;"><p> ->  Angabe eines primären und sekundären Speicherorts.</p></div><div style="text-indent: 20px;"><b>Felder: Anlegen</b></div><div style="text-indent: 20px;"><p> ->  Legt ein neues Archivfeld mittels eines Dialogs an.</p></div><div style="text-indent: 20px;"><b>Felder: Bearbeiten</b></div><div style="text-indent: 20px;"><p> ->  Bearbeitet ein Archivfeld mittels eines Dialogs.</p></div><div style="text-indent: 20px;"><b>Felder: Löschen</b></div><div style="text-indent: 20px;"><p> ->  Löscht das fokussierte Feld.
</p></div><div style="text-indent: 20px;"><b>Felder: Aktualisieren</b></div><div style="text-indent: 20px;"><p> ->  Aktualisiert die Ansicht der Felder.
</p></div><div style="text-indent: 20px;"><b>Profile: Anlegen</b></div><div style="text-indent: 20px;"><p> ->  Legt ein neues Profil mittels eines Dialogs an.</p></div><div style="text-indent: 20px;"><b>Profile: Bearbeiten</b></div><div style="text-indent: 20px;"><p> ->  Bearbeitet ein Profil mittels eines Dialogs.</p></div><div style="text-indent: 20px;"><b>Profile: Kopieren</b></div><div style="text-indent: 20px;"><p> ->  Kopiert das Profil und legt die Kopie als neuen Eintrag der Liste an.
</p></div><div style="text-indent: 20px;"><b>Profile: Entfernen</b></div><div style="text-indent: 20px;"><p> ->  Entfernt das fokussierte Profil. 
</p></div><div style="text-indent: 20px;"><b>Profile: Aktualisieren</b></div><div style="text-indent: 20px;"><p> ->  Aktualisiert die Ansicht der Profile.
</p></div><div style="text-indent: 20px;"><b>Optionen: read only</b></div><div style="text-indent: 20px;"><p> ->  Erlaubt nur lesenden Zugriff auf das Archiv. Es können keine Änderungen vorgenommen werden.</p></div><div style="text-indent: 20px;"><b>Optionen: Erweiterte Protokollierung</b></div><div style="text-indent: 20px;"><p> ->  Aktiviert die erweiterte Protokollierung. Hierbei kann es zu Performanceeinbußen kommen.</p></div><div style="text-indent: 20px;"><b>Optionen: Leerzeichen entfernen</b></div><div style="text-indent: 20px;"><p> ->  Entfernt bei Textfeldern Leerzeichen am Anfang und Ende.</p></div><div style="text-indent: 20px;"><b>Optionen: Maximale Belegzahl</b></div><div style="text-indent: 20px;"><p> ->  Definiert, wie viele Belege vom Archiv maximal zurückgegeben werden. Standard Wert sind 10.000.</p></div><div style="text-indent: 20px;"><b>Optionen: Maximale Anzahl</b></div><div style="text-indent: 20px;"><p> ->  Definiert, wie viele Dokumente maximal durch die Volltextsuche zurückgegeben werden, Standard Wert sind 100.000.</p></div><div style="text-indent: 20px;"><b>Image Server: Verarbeitung von Seiten beschränken</b></div><div style="text-indent: 20px;"><p> ->  Beschränkt die Anzahl von Seiten die vom Image Server bearbeitet werden.
</p></div><div style="text-indent: 20px;"><b>Image Server: Texterkennung aktivieren</b></div><div style="text-indent: 20px;"><p> ->  Aktiviert die Texterkennung für PDF/A Dateien.
</p></div><div style="text-indent: 20px;"><b>Image Server: Bildaufbau aktivieren</b></div><div style="text-indent: 20px;"><p> ->  Aktiviert den Bildaufbau durch den Image Server.
</p></div><div style="text-indent: 20px;"><b>OCR Server: Archiv für Volltexterkennung aktivieren</b></div><div style="text-indent: 20px;"><p> ->  Aktiviert den OCR Server für die Volltext Erkennung.
</p></div><div style="text-indent: 20px;"><b>OCR Server: Koordinaten speichern</b></div><div style="text-indent: 20px;"><p> ->  Die Koordinaten der Wörter werden gespeichert.
</p></div><div style="text-indent: 20px;"><b>OCR Server: Zeichen im OCR Text entfernen</b></div><div style="text-indent: 20px;"><p> ->  Entfernt alle Wörter die eine bestimmte Zeichenzahl unterschreiten.
</p></div><div style="text-indent: 20px;"><b>Statistik</b></div><div style="text-indent: 20px;"><p> ->  Hier werden verschiedene Statistiken zu dem Archiv angezeigt. </p></div><div style="text-indent: 20px;"><b>Volltext</b></div><div style="text-indent: 20px;"><p> ->  Zeigt Informationen zum Volltext an. </p></div><p></p><span style="color: #808080;"> Ende Funktions-ID:  636 Modul-ID:88</span><hr/>
