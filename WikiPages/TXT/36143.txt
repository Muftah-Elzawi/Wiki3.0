<br><br /><span style="color: #808080;"> Beginn Funktions-ID:  661 Modul-ID:91</span><br><br /><h3>Benutzer bearbeiten</h3><p>Öffnet einen Dialog in dem der Benutzer bearbeitet werden kann.
</p><br><br /><div style="text-indent: 20px;"><img src="/wiki/filemanager/images\Funktionskatalog\91_661.png" alt="" /><br /></div><br><br /><div style="text-indent: 20px;"><h4>Parameter der Funktion</h4></div><div style="text-indent: 20px;"><b>Einstellungen: Informationen</b></div><div style="text-indent: 20px;"><p> ->  Anzeige der persönlichen Informationen des Benutzers. Zum Beispiel Name, Benutzername und Organisation.</p></div><div style="text-indent: 20px;"><b>Einstellungen: Kennwort Änderung</b></div><div style="text-indent: 20px;"><p> ->  Legt fest, dass bei der nächsten Anmeldung das Kennwort geändert werden muss.</p></div><div style="text-indent: 20px;"><b>Einstellungen: 2-Faktor Authentifizierung</b></div><div style="text-indent: 20px;"><p> ->  Aktiviert die 2-Faktor Authentifizierung.</p></div><div style="text-indent: 20px;"><b>Einstellungen: Aufgabenempfang aktivieren</b></div><div style="text-indent: 20px;"><p> ->  Aktiviert die Option, dass der Benutzer Aufgaben empfangen kann.</p></div><div style="text-indent: 20px;"><b>Einstellungen: Änderungen durch Benutzer sperren</b></div><div style="text-indent: 20px;"><p> ->  Verhindert, dass der Benutzer selbst seine Daten ändern kann.</p></div><div style="text-indent: 20px;"><b>Gruppen: Hinzufügen</b></div><div style="text-indent: 20px;"><p> ->  Fügt den Benutzer zu weiteren Gruppen mittels eines Dialogs hinzu.</p></div><div style="text-indent: 20px;"><b>Gruppen: Entfernen</b></div><div style="text-indent: 20px;"><p> ->  Entfernt den Benutzer aus der Gruppe.</p></div><div style="text-indent: 20px;"><b>Gruppen: Aktualisieren</b></div><div style="text-indent: 20px;"><p> ->  Aktualisiert die Ansicht der Gruppen.</p></div><div style="text-indent: 20px;"><b>Lizenz: Hinzufügen</b></div><div style="text-indent: 20px;"><p> ->  Vergibt weitere Lizenzen mittels eines Dialogs an den Benutzer.</p></div><div style="text-indent: 20px;"><b>Lizenz: Entfernen</b></div><div style="text-indent: 20px;"><p> ->  Entzieht dem Benutzer die fokussierte Lizenz.</p></div><div style="text-indent: 20px;"><b>Lizenz: Aktualisieren</b></div><div style="text-indent: 20px;"><p> ->  Aktualisiert die Ansicht der Lizenzen.</p></div><div style="text-indent: 20px;"><b>Benutzerinformation: Email</b></div><div style="text-indent: 20px;"><p> ->  Der Benutzer wird per Email über Änderungen an seinen Daten informiert.</p></div><div style="text-indent: 20px;"><b>Benutzerinformation: SMS</b></div><div style="text-indent: 20px;"><p> ->  Der Benutzer wird per SMS über Änderungen an seinen Daten informiert.</p></div><div style="text-indent: 20px;"><b>Status</b></div><div style="text-indent: 20px;"><p> ->  Zeigt Statusinformationen zum Benutzer an.</p></div><p></p><span style="color: #808080;"> Ende Funktions-ID:  661 Modul-ID:91</span><hr/>
