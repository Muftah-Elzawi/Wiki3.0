<br><br /><span style="color: #808080;"> Beginn Funktions-ID:  430 Modul-ID:79</span><br><br /><h3>Server bearbeiten </h3><p>Öffnet einen Dialog um einen Server zu bearbeiten.
</p><br><br /><div style="text-indent: 20px;"><img src="/wiki/filemanager/images\Funktionskatalog\79_430.png" alt="" /><br /></div><br><br /><div style="text-indent: 20px;"><h4>Parameter der Funktion</h4></div><div style="text-indent: 20px;"><b>Reiter Server: Organisation</b></div><div style="text-indent: 20px;"><p> ->  Der Server wird dieser Organisation zugeordnet.</p></div><div style="text-indent: 20px;"><b>Reiter Server: Lizenzbenutzer</b></div><div style="text-indent: 20px;"><p> ->  Name des Users der die Lizenzen für die Jobserver Module hält.</p></div><div style="text-indent: 20px;"><b>Reiter Server: Benennung</b></div><div style="text-indent: 20px;"><p> ->  Bezeichnung des Servers. Diese Bezeichnung hat keine technische Relevanz.</p></div><div style="text-indent: 20px;"><b>Reiter Server: Servername</b></div><div style="text-indent: 20px;"><p> ->  Technischer Name des Servers.</p></div><div style="text-indent: 20px;"><b>Reiter Server: Beschreibung</b></div><div style="text-indent: 20px;"><p> ->  Es kann eine Beschreibung für den Server angelegt werden.</p></div><div style="text-indent: 20px;"><b>Reiter Server: aktiver Server</b></div><div style="text-indent: 20px;"><p> ->  Der Server wird als aktiv markiert. Es kann nur auf aktive Server zugegriffen werden.</p></div><div style="text-indent: 20px;"><b>Reiter Optionen: Jobserver Intervall</b></div><div style="text-indent: 20px;"><p> ->  In diesem zeitlichen Abstand überprüft der Server die Aufgaben.</p></div><div style="text-indent: 20px;"><b>Reiter Optionen: Anzahl Einträge in Historie</b></div><div style="text-indent: 20px;"><p> ->  Begrenzt die Anzahl der Einträge in der Serverhistorie auf die eingegebene Anzahl. Anzahl =0 bedeutet unbegrenzte Einträge. Eine Begrenzung der Einträge kann eine Verbesserung der Ladezeiten bewirken.</p></div><div style="text-indent: 20px;"><b>Reiter Optionen: Leistungsdaten sammeln</b></div><div style="text-indent: 20px;"><p> ->  Die Leistungsdaten werden gesammelt und aus die Festplatte geschrieben. Diese Funktion kann im Fall von Performance Problemen zur Analyse aktiviert werden, andernfalls ist es besser diese Funktion deaktiviert zu lassen.
</p></div><p></p><span style="color: #808080;"> Ende Funktions-ID:  430 Modul-ID:79</span><hr/>
