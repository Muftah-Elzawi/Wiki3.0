<br><br /><span style="color: #808080;"> Beginn Funktions-ID:  405 Modul-ID:76</span><br><br /><h3>Aufgabe bearbeiten</h3><p>Öffnet einen Dialog zum bearbeiten einer Aufgabe im Reiter Aktionen.
</p><br><br /><div style="text-indent: 20px;"><img src="/wiki/filemanager/images\Funktionskatalog\76_405.png" alt="" /><br /></div><br><br /><div style="text-indent: 20px;"><h4>Parameter der Funktion</h4></div><div style="text-indent: 20px;"><b>Allgemein</b></div><div style="text-indent: 20px;"><p> ->  Allgemeine Informationen zur Jobserver Aufgabe. Zum Beispiel Organisation, Server oder Aufgaben ID.</p></div><div style="text-indent: 20px;"><b>Aktionen: Modul hinzufügen</b></div><div style="text-indent: 20px;"><p> ->  Fügt ein neues Modul mittels eines Dialogs hinzu.</p></div><div style="text-indent: 20px;"><b>Aktionen: Modul bearbeiten</b></div><div style="text-indent: 20px;"><p> ->  Bearbeitet ein Modul mittels eines Dialogs.
</p></div><div style="text-indent: 20px;"><b>Aktionen: Modul konfigurieren</b></div><div style="text-indent: 20px;"><p> ->  Öffnet einen Dialog mit dem das fokussierte Modul konfiguriert werden kann.
</p></div><div style="text-indent: 20px;"><b>Aktionen: Modul löschen</b></div><div style="text-indent: 20px;"><p> ->  Entfernt das fokussierte Modul.</p></div><div style="text-indent: 20px;"><b>Aktionen: Ansicht aktualisieren</b></div><div style="text-indent: 20px;"><p> ->  Aktualisiert die Ansicht der Aufgabenliste.</p></div><div style="text-indent: 20px;"><b>Aktionen: Modul ausführen</b></div><div style="text-indent: 20px;"><p> ->  Führt das Modul aus manuell aus und Speichert das Protokoll im Admin Log.
</p></div><div style="text-indent: 20px;"><b>Aktionen: nach oben/unten verschieben</b></div><div style="text-indent: 20px;"><p> ->  Verschiebt das Modul in der Liste um eine Stelle nach oben/unten.</p></div><div style="text-indent: 20px;"><b>Aktionen: Positionen neu berechnen</b></div><div style="text-indent: 20px;"><p> ->  Berechnet die Position der Aktionen neu. Ggf. wird die Nummerierung verändert.</p></div><div style="text-indent: 20px;"><b>Aktionen: Modul aktivieren/deaktivieren</b></div><div style="text-indent: 20px;"><p> ->  Markiert das fokussierte Modul als aktiv/inaktiv. Abhängig vom Ausgangszustand kann ist für ein Modul der De-/Aktivieren Button nicht verfügbar.
</p></div><div style="text-indent: 20px;"><b>Aktionen: Modul kopieren</b></div><div style="text-indent: 20px;"><p> ->  Kopiert die Konfiguration der Aktion.</p></div><div style="text-indent: 20px;"><b>Aktionen: Modul verschieben</b></div><div style="text-indent: 20px;"><p> ->  Das Modul wird in eine andere Aufgabe verschoben.  Das Modul ist anschließend nicht mehr in der aktuellen Aufgabe vorhanden.
</p></div><div style="text-indent: 20px;"><b>Aktionen: Modul exportieren</b></div><div style="text-indent: 20px;"><p> ->  Exportiert das fokussierte Modul.</p></div><div style="text-indent: 20px;"><b>Aktionen: Modul importieren</b></div><div style="text-indent: 20px;"><p> ->  Importiert eine Modul Date.</p></div><div style="text-indent: 20px;"><b>Aktionen: Beschreibung ein-/ausblenden</b></div><div style="text-indent: 20px;"><p> ->  Blendet die Modulbeschreibung ein/aus.</p></div><div style="text-indent: 20px;"><b>Aktionen: neue Version erstellen</b></div><div style="text-indent: 20px;"><p> ->  Erstellt eine neue Version dieser Konfiguration.
</p></div><div style="text-indent: 20px;"><b>Aktionen: Versionen anzeigen</b></div><div style="text-indent: 20px;"><p> ->  Zeigt die Versionen eines Moduls an.</p></div><div style="text-indent: 20px;"><b>Zeitsteuerung: hinzufügen</b></div><div style="text-indent: 20px;"><p> ->  Legt eine neue Zeitsteuerung mittels eines Dialogs an.
</p></div><div style="text-indent: 20px;"><b>Zeitsteuerung: bearbeiten</b></div><div style="text-indent: 20px;"><p> ->  Bearbeitet eine Zeitsteuerung mittels eines Dialogs.</p></div><div style="text-indent: 20px;"><b>Zeitsteuerung: löschen</b></div><div style="text-indent: 20px;"><p> ->  Entfernt die fokussierte Zeitsteuerung.
</p></div><div style="text-indent: 20px;"><b>Zeitsteuerung: aktualisieren</b></div><div style="text-indent: 20px;"><p> ->  Aktualisiert die Ansicht der Ergebnisliste.
</p></div><div style="text-indent: 20px;"><b>Zeitsteuerung: zurücksetzen</b></div><div style="text-indent: 20px;"><p> ->  Die Zeitsteuerung wird zurückgesetzt.</p></div><div style="text-indent: 20px;"><b>Historie: Ansicht Export</b></div><div style="text-indent: 20px;"><p> ->  Die aktuelle Ansicht wird exportiert.
</p></div><div style="text-indent: 20px;"><b>Historie: CSV Export</b></div><div style="text-indent: 20px;"><p> ->  Exportiert Die aktuelle Ansicht als CSV Datei.</p></div><div style="text-indent: 20px;"><b>Historie: löschen</b></div><div style="text-indent: 20px;"><p> ->  Es werden alle Einträge aus der Historie entfernt.
</p></div><div style="text-indent: 20px;"><b>Einstellungen: Protokollierung</b></div><div style="text-indent: 20px;"><p> ->  Gibt an ob und was protokolliert werden soll. Es können die Informationen, Warnungen und Fehler protokoliert werden oder auch gar nichts.
</p></div><div style="text-indent: 20px;"><b>Einstellungen: Aufgabe beenden</b></div><div style="text-indent: 20px;"><p> ->  Die Aufgabe wird beendet falls die Bearbeitung die eingegebene Zeitspanne überschreitet.</p></div><div style="text-indent: 20px;"><b>Einstellungen: Aufgabe in separatem Prozess</b></div><div style="text-indent: 20px;"><p> ->  Die Aufgabe würde in einem separaten Prozess ausgeführt.</p></div><p></p><span style="color: #808080;"> Ende Funktions-ID:  405 Modul-ID:76</span><hr/>
